# Todo App

## Overview
This is a basic frontend-only todo application built with HTML, CSS, and JavaScript. It utilizes IndexedDB for persistent storage and falls back to localStorage for browsers that do not support IndexedDB.

## Features
- Add, edit, and delete todos
- Mark todos as complete
- Persistent storage using IndexedDB with localStorage as a fallback

## Usage
1. Open `index.html` in your browser.
2. Use the input field to add new todos.
3. Click on the todos to mark them as complete or to edit them.
4. Todos will be saved in your browser's storage.

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Note
This project does not use Node.js or npm dependencies. It is a standalone front-end application suitable for learning purposes.
